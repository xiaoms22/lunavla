# v1.0 script archive

These scripts generated the initial completion reports, portfolio packs, static media, and ad hoc comparisons. They depend on pre-v1.1 names, legacy checkpoint paths, transient `outputs/` files, or pre-correction evidence assumptions.

They are retained for provenance and are not supported entry points. Do not run them from this archive as v1.1 tooling.

Current commands remain in the repository-root `scripts/` directory and use strict configs, `checkpoint.json`, core `RunManifest`, controlled experiment summaries, and `results/v1.1` snapshots.
